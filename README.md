# Hefei Metro Lines and Stations Map

An interactive web map showing Hefei Metro lines and station locations.

## View the Map

- Main page: https://liteng16.github.io/hefei-metro-map/
- Muted map: https://liteng16.github.io/hefei-metro-map/muted/
- Colorful map: https://liteng16.github.io/hefei-metro-map/colorful/
- Gaode map: https://liteng16.github.io/hefei-metro-map/amap/

## Versions

- `muted/`: a low-saturation basemap version that makes the metro lines stand out more clearly.
- `colorful/`: a full-color OpenStreetMap basemap version with richer city context.
- `amap/`: a Gaode / AutoNavi basemap version using GCJ-02 coordinates, intended for better access in mainland China.

## Features

- Different metro lines are shown in different colors.
- Station locations are shown as point markers.
- Station names can be shown or hidden.
- Users can search for stations.
- The map works on both desktop and mobile browsers.

## Data Source

Metro line and station data are derived from OpenStreetMap.
